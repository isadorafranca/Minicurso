## 🌿 Sobre

Este não é só um curso, é um pontaq pé inicial para quem quer come;ar na área de dados.

O **Do Mar ao Monitor** nasce da necessidade de traduzir dados ambientais em decisões reais.

Aqui, você aprende a pegar dados dispersos
organizar
interpretar
e transformar em evidência para conservação

---

## 🎯 O que você vai aprender

* acessar dados ambientais reais (GBIF, clima, oceano, genes)
* estruturar bancos científicos do zero
* limpar dados e evitar erros comuns
* explorar padrões ecológicos
* construir visualizações simples e úteis
